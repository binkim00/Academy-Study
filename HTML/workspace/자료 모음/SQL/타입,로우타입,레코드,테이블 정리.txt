======================== PL/SQL 변수 타입 핵심 비교표 ========================

구분            | 용도                           	| 사용하는 이유                          		| 선언 방식                               	  | 저장 형태
-------------- |------------------------------------|---------------------------------------------------|------------------------------------------|-------------
%TYPE         | 컬럼의 데이터 타입 참조            | 컬럼 타입을 정확히 따르기 위해         	| 변수명 테이블명.컬럼명%TYPE            | 단일 값
%ROWTYPE   | 테이블 한 행 전체 참조       	| 여러 컬럼을 한 번에 다루기 위해         	| 변수명 테이블명%ROWTYPE               | 복합 구조 (여러 컬럼)
RECORD       | 사용자 정의 복합 타입        	| 원하는 필드만 직접 지정해서 쓰고 싶을 때 	| TYPE 이름 IS RECORD (...) 선언 후 사용 | 복합 구조 (사용자 정의)
TABLE TYPE   | 배열(리스트)처럼 여러 값 저장	| 여러 행/값을 반복 처리하고 싶을 때       	| TYPE 이름 IS TABLE OF ... INDEX BY ...  | 배열/리스트
